Detailed Description

template<typename T>
class Scc::CountMin< T >

CountMin-Sketch can be used to estimate the frequencies of items.

Estimation error of a given item \(a\) can be bounded by \(\frac{n - f_a}{m}\) where \(n\) is the length of the data stream, \(f_a\) is the actual frequency of \(a\), and \(m\) is the size of buffer.

Success probability is at least \(1 - 2^{-d}\).
