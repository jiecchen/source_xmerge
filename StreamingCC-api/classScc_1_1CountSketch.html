Detailed Description

template<typename T>
class Scc::CountSketch< T >

Count-Sketch can be used to estimate the frequencies of items.

Estimation error of a given item \(a\) can be bounded by \(\frac{\|\mathbf{f}_{-a}\|_2}{\sqrt{m}}\) where \(\mathbf{f}_{-a}\) is the frequency vector without \(a\) (i.e. drop coorinate of \(a\) from the data stream frequency vector), \(m\) is the size of buffer.

Success probability is at least \(1 - 2^{-d}\).
