Detailed Description

template<typename T>
class Scc::DistinctCounter< T >

Class template to count number of distinct elements over a data stream.

The relative error of the estimation is bounded by \(\frac{1}{\sqrt{\text{buf_size}}}\). Success probility is at least \(1 - 2^{-d}\)
