Detailed Description

template<typename T>
class Scc::F2< T >

Class for estimating the 2nd frequency moment of a weighted data stream.

The 2nd moment of a vector \(\textbf{x}\) is defined as \(\|\textbf{x}\|_2^2\). The 2nd moment of frequency vector of a data stream can be used to mearsure how skew the data is.

This class works for weighted data stream as well (i.e. you can assign a weight for each item). When weight is omitted, \(1\) is used by default.

The relative error of estimation is bounded by \(\frac{1}{\sqrt{buf\_size}}\). Success probability is at least \(1 - 2^{-d}\).
