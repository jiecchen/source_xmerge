Detailed Description

template<typename T>
class Scc::MisraGries< T >

Class template for Misra-Gries algorithm.

Misra-Gries algorithm with parameter \(k\) (buffer size) can be used to find heavy hitters whose frequencies are larger than \(\frac{n}{k}\), here \(n\) is the total number of items.

For any item, Misra-Gries also gives an estimation of its frequency with error bounded by \(\frac{n}{k}\).
