Detailed Description

template<typename T>
class Scc::ReservoirSampling< T >

Sampling over a data stream via Reservoir Sampling.

  • For sampling with replacement, both weighted an unweighted data streams are supported.
  • For sampling without replacement, currently only support unweighted data stream.
